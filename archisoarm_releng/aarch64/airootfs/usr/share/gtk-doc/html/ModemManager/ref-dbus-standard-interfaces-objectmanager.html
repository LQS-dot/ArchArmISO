org.freedesktop.DBus.ObjectManager

The Manager object exported at the org.freedesktop.ModemManager1 bus name implements the standard org.freedesktop.DBus.ObjectManager interface. This interface, included in rev. 0.17 of the DBus specification, allows a generic way to control the addition and removal of Modem objects, as well as the addition and removal of interfaces in the given objects.
