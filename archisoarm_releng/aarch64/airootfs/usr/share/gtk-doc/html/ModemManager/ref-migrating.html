Part V. Migrating from ModemManager 0.6 to ModemManager 1.0

ModemManager 1.0 is a new major version of ModemManager that breaks both API and ABI compared to previous versions. These changes allow better managing new types of devices (e.g. those with multiple capabilities), or those not based in AT commands for operation (e.g. QMI or MBIM modems).

This section provides an introduction to the changes done in the DBus interface with respect to the main operations performed with modems through ModemManager.
