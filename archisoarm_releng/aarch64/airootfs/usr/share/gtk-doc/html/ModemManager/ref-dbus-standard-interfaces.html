org.freedesktop.DBus.Properties

All objects (Manager, Modems, Bearers, SIMs, SMSs) exported at the org.freedesktop.ModemManager1 bus name implement the standard org.freedesktop.DBus.Properties interface. Objects implementing this interface provide a common way to query for property values and also a generic signal to get notified about changes in those properties.
